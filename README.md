# quote‑fetcher

A one‑liner CLI written in Python that prints a random quote.

## Usage
```bash
# Install dependencies (or use a virtual‑env)
python -m pip install -r requirements.txt

# Run the tool
python main.py
```

## Why a tiny project?
* Demonstrates **instant init** – a single file does the job.
* Includes **comprehensive validation** in the CI workflow (lint, tests, dependency pins).
* Shows **secure GitHub Actions** practices: action pinning, least‑privilege token, no secret interpolation, OIDC‑ready, etc.
